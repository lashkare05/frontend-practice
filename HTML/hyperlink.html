Hyperlink

The anchor tag is used to create hyperlinks. Hyperlinks allow users to navigate from on page, file, email, or location to another.

  • <a></a> = Anchor tag
  • href = Destination URL
  • Visit Website = Clikable text

  1. href Attribute

    href stands for Hypertext Reference.
    It specifies where the link should go.



    < a href="www.google.com"> Google Search </a>

Absolute vs Relative Links

Absolute URL : Full web address.

<a href="https://www.google.com">Google</a>

Relative URL : Links to files within the project.

<a href="about.html">About</a>

You can wrap an anchor tag (<a>) around almost any html element, making that element clickable .

  1. Text (most common)
    <a href="https://example.com">Visit Website</a>
  2. Image
    <a href="https://example.com"><img src="logo.png" alt="Logo" > </a>
    Clicking the image opens the link.
  3. Audio
    <a href="song.mp3"><Download Audio</a>
    Or make an audio player clickable.
    <a href="song.mp3">< <audio controls> <source src="song.mp3" type="audio/mpeg" > </audio> </a>
  4. Video
    <a href="video.mp4"><video width="300" controls> <source src="video.mp4" type="video/mp4" ></video> </a>
  5. Buttons
    <a href="contact.html"> <button>Contact Us</button></a>

    although in modern HTML, it's usually better style the anchor tag(<a>) like a button. I mean that if clicking something. takes the user another page or url. You should use anchor tag even if it looks like a button.
    <a href="contact.html"> <button>Contact Us</button></a>     Less preferred .
    Here, a button is placed inside a link. It works, but it's not the best semantic html.

    Prefered : Use an anchor tag and mix it look like a button using css.

    <a href="contact.html" class="btn"> Contact Us</a> 
                            .btn{
                            display: inline-block;
                            padding: 10px 20px;
                            background-color: blue;
                            color: white;
                            text-decoration: none;
                            border-radius: 5px;
                        }
    Contact us looks like a button, but it's actually a link.

    When to use anchor tag vs button ?

    use anchor tag when :

    • Going to another page
    • Opening a website
    • Downloading a file
    • Sending an email
    • Calling a phone number

    Use button when :
    • Submitting a form ,
    • opening a popup/modal
    • running javascript
    • showing/hiding content

    Navigation somewhere = anchor tag
    perform an action = button

    • Go to homepage --> <a>
    • Submit form --> <button>
    • Download Resume --> <a>
    • Show Password --> <button>
    This distinction is important because browsers, screen readers, and search engines understand the purpose of the element based on whether its an anchor tag or a button.

  6. Entire Section / Cards
    <a href="product.html"> <div class="card"><h2>Product Name</h2> <p>Product Description</p> </div> </a>

    The whole card becomes clickable . After enclosing whole section or car in anchor tag that section or car become clickable .

Rule to remember

An ankle tag can wrap

  • Text
  • Images
  • Icons
  • Buttons
  • Audio elements
  • Video elements
  • Divs, cards ,sections, and many other html elements/li>

Its purpose is simple : make the wrapped content clickable and Navigate to another resource, page, file, email address, phone number, or section of the page .
